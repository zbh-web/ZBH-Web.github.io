# 赛博朋克风格个人网站

一个具有赛博朋克风格的GitHub个人网站，包含霓虹灯效果、故障艺术、终端界面和互动元素。

## 功能特性

### 🎨 视觉效果
- 霓虹蓝、粉、紫色调
- 故障艺术文字效果
- 扫描线动画
- 网格背景
- 脉冲光环

### 🖥️ 交互元素
- 动态技能条
- 数字计数器
- 终端打字机效果
- 可交互控制台
- 鼠标跟随效果

### 🔊 音效系统
- 背景环境音乐
- 按钮点击音效
- 音频控制面板

### 📱 响应式设计
- 自适应各种屏幕尺寸
- 移动端优化
- 触摸屏支持
